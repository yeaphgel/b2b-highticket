## ✨ 核心特性

### 1. 🎯 Skill 插件安装模式（主要亮点）

**之前**：需要启动 Express 本地服务，配置 Webhook 和 API  
**现在**：一条命令安装到 OpenClaw/Hermes，平台自动识别并启用

```bash
# OpenClaw 一键安装
bash <(curl -fsSL https://raw.githubusercontent.com/yeaphgel/b2b-highticket/main/install-openclaw.sh)

# Hermes 一键安装  
bash <(curl -fsSL https://raw.githubusercontent.com/yeaphgel/b2b-highticket/main/install-hermes.sh)
```

**优势**：
- ✅ 无需管理端口和服务进程
- ✅ 自动依赖安装和环境检测
- ✅ 向量索引自动构建
- ✅ 支持环境变量和交互式配置

### 2. 🧠 AI 实时教练引擎

通过自然语言对话，提供：
- **问题诊断**：快速定位销售话术和方法问题
- **话术推荐**：基于知识库的实时应对建议
- **客户分析**：竞品应对、报价策略、行业案例

**使用示例**：
```
"客户说价格太贵怎么办？"
@clover → 返回 3 种应对话术 + 案例参考

"帮我分析 A 客户的进展"
@clover → 调取客户档案 + 阶段建议 + 下周跟进提示
```

### 3. 📊 十维度能力评分体系

AI 自动评估销售在 8 个关键节点的表现：

| 维度 | 描述 | 初级权重 | 高级权重 |
|------|------|---------|---------|
| 破冰 | 初次接触的融洽度 | 15% | 10% |
| 需求识别 | 痛点挖掘深度 | 20% | 15% |
| 价值传达 | 方案说服力 | 15% | 10% |
| 信任建立 | 共识达成程度 | 15% | 10% |
| 信任塑造 | 关系深化 | 10% | 25% |
| 定制解决 | 方案个性化程度 | 10% | 20% |
| 异议处理 | 反对意见应对 | 10% | 5% |
| 促成交易 | 成交推动力 | 5% | 5% |
| 关系维护 | 售后管理 | 5% | 15% |
| 跟进钩子 | 下次触发点 | 5% | 15% |

**智能特性**：
- 权重自动适应销售等级（L1 新人 ↔ L5 大师）
- 对标团队销冠，找出差距维度
- 每周自动计算进度变化

### 4. 🏆 GROW 教练引擎

基于 Goal-Reality-Explore-Will 框架的个性化教练：

```
目标(Goal)   → 本周聚焦点（基于低分维度）
现状(Reality) → 对标数据（与销冠对比）
探索(Explore) → 3 个具体改进动作
承诺(Will)   → 下周验证指标
```

**自动触发**：每周自动生成个性化教练分析

### 5. 🎙️ 通话自动复盘（Hermes 专属）

#### 工作流程
```
通话开始 → Hermes 录音
   ↓
通话结束 → 自动转写（准确率 95%+）
   ↓
AI 分析 → 提取痛点、需求、承诺
   ↓
十维评分 → 对标历史和销冠
   ↓
GROW 教练 → 自动生成改进方案
   ↓
用户反馈 → 确认并保存记录
```

#### 自动识别内容
- 🎯 **客户需求**：隐性需求、真实痛点、优先级
- 💬 **话术质量**：开场白、提问深度、价值论述清晰度
- 📈 **进展信号**：客户热度、决策推进、风险识别
- 🏆 **对标反馈**：与销冠的相似话术、改进建议

### 6. 💾 客户档案记忆系统

自动记录和管理客户信息：

```
客户档案包括：
├── 基本信息（公司、行业、联系人）
├── 销售阶段（初次接触 → 商务谈判 → 签单）
├── 痛点和需求
├── 跟进历史（时间线日志）
├── 下一步行动
└── 相关承诺和截止期
```

**自动提示**：
- 跟进客户时自动加载历史背景
- 完成对话后提示"要更新客户档案吗？"
- 支持快速查询整个团队的客户列表

### 7. 📚 语义搜索知识库

使用 ARK API 的向量索引，支持：
- 话术库搜索（竞品应对、报价技巧）
- 案例库搜索（行业解决方案、成功案例）
- 销售方法论（Clover 八关、SPIN、Sandler、RAIN）
- 自定义知识上传

**使用示例**：
```
"有没有 SaaS 行业的 ROI 计算方法？"
@clover → 搜索知识库返回相关案例和话术
```

## 🔧 技术改进

### Skills 目录自动检测

支持多层级识别机制：

```bash
1. openclaw/hermes CLI 命令
2. OPENCLAW_SKILLS_DIR / HERMES_SKILLS_DIR 环境变量  
3. ~/.agents/skills/  （最常见默认路径）
4. ~/.openclaw/skills/ 或 ~/.hermes/skills/
5. 交互式提示用户输入
```

### 环境变量智能配置

```bash
# 优先级：环境变量 > 交互输入 > 默认值

ARK_API_KEY=xxx npm install-openclaw.sh
# 自动读取环境变量，无需交互
```

### 向量索引自动构建

```bash
# 安装完成后自动运行
cd scripts && node index.js
# 首次构建约需 1-3 分钟
```

## 🐛 已知问题和限制

### 当前版本（v1.0.0）

1. **Hermes 语音转写**：准确率最高 95%，技术术语可能误识别
   - **解决方案**：v1.1 版支持自定义术语库

2. **知识库构建时间**：首次构建 1-3 分钟
   - **优化**：后续版本支持增量更新

3. **十维度评分**：基于通话内容，需要充分的对话样本（建议 5+ 通话）
   - **建议**：新员工前两周可能评分波动较大，正常现象

### 计划改进（v1.1+）

- 🔜 原生 UI 组件（不仅仅是 Skill）
- 🔜 实时通话 AI 建议（Hermes 增强）
- 🔜 定制化维度和权重配置
- 🔜 批量客户档案导入
- 🔜 与 Salesforce、HubSpot 的原生集成
