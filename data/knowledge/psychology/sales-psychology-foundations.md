# 销售心理学基础：Cialdini 影响力原理与 Clover 融合

**基础理论**：罗伯特·西奥迪尼（Robert Cialdini）《影响力》+ 丹尼尔·卡尼曼《思考快与慢》

**核心**：所有销售成交都遵循心理学规律，而非技巧堆砌

## Cialdini 的 6 大影响力原理

### 原理 1：互惠原则（Reciprocity）

**心理学解释**：
人有一种强烈的心理需要：别人对我好，我就要对别人好。这不是理性，是本能。

**在销售中的应用**

| 环节 | 应用方式 | 话术例子 |
|------|---------|---------|
| **G1 破冰** | 先给价值，建立"债务" | "我看了你们的账号，整理了一份竞品分析，先给你参考" |
| **G2 诊断** | 深度倾听，展现关心 | "你说得很对，我记下来了。你还遇到过…吗？" |
| **G7 跟单** | D1 免费送诊断报告 / D3 送行业触点 / D5 送案例 | 不催单，先送价值 |

**为什么有效**：
- 客户接收了价值 → 产生了"欠债感"
- 这种欠债感会自动推动他们想要"还债"（通常的还债方式就是……成交）

**Cialdini 的经典研究**：
在一个餐厅实验中，如果服务员给客户一个薄荷糖，客户的小费金额会增加 3-5%。原因很简单：互惠原则。

---|---------|---------|
| **G1** | 小小的承诺 | "你同意我们今天花 30 分钟聊吗？" |
| **G2** | 让客户承诺问题的存在 | "所以你的情况是…这个问题确实存在，对吗？" |
| **G3** | 让客户承诺问题的严重性 | "如果继续这样，这个问题对你来说有多紧急？非常紧急，对吗？" |
| **G4** | 让客户承诺改变意愿 | "你是想现在解决，还是继续这样？" |
| **G6** | 小步成交 | Sandler 的"小步承诺法则" |

**为什么有效**：

Cialdini 的著名实验：
- A 组：问"你会参加明天的环保活动吗？" → 17% 参加率
- B 组：先问"你关心环保吗？" → 客户说"是" → 再问"你会参加…吗？" → 76% 参加率

区别在哪？B 组通过第一个"是"建立了身份认同。一旦你说"我关心环保"，你就被迫要采取环保的行动。

**Clover 的融合**：
- G4 引导承诺中的每一步都是小小的"是"
- 客户每说一个"是"，他的身份认同就向"决定者"靠近
- 到了 G6，最后的"是"就变成了顺其自然

### 原理 3：社会证明（Social Proof）

**心理学解释**：
在不确定的时候，人倾向于看别人在做什么，然后跟风。这叫"群体智慧"或"羊群效应"。

**在销售中的应用**

| 环节 | 应用方式 | 话术例子 |
|------|---------|---------|
| **G1 破冰** | 提前建立权威 | "我们服务了 500+ 企业，其中有…" |
| **G5 价值展示** | 用案例，特别是"和你们类似的企业" | "跟你们情况类似的餐饮连锁，3 个月从 0 粉到 10K 粉，带来 50+ 询价" |
| **G7 跟单** | D5 "案例证明" | "我们一个客户本周的内容审批通过了，数据出来分享给你" |

**为什么有效**：

Cialdini 的"受害者实验"：
- 有人在街上倒下，只有 1-2 人围观时，很少人会帮忙（责任分散）
- 但如果有 10 个人围观且有人在帮忙，更多人会加入

销售中的应用：
```
销售说："我们的方案很好"
→ 客户心想："你说好，肯定想卖东西"（防御）

销售说："我们服务的 100 家企业，有 94% 续费了"
→ 客户心想："这么多人都在用，应该没问题"（社会证明）
```

**最强形式的社会证明**：
不是"100 个客户在用"，而是"和你们一样的企业在用"

```
弱：我们有很多客户
中：我们有 500+ 企业客户
强：你们的竞品 XXX 也在用我们的方案
最强：跟你们一样是年销 ¥1000 万的电商企业，他们用我们的方案后…
```

### 原理 4：权威性（Authority）

**心理学解释**：
人会自动相信来自权威人士的信息。不需要全部信息，只需要权威背书。

**在销售中的应用**

| 环节 | 应用方式 | 话术例子 |
|------|---------|---------|
| **G1 破冰** | 建立专家身份 | "我看了你们的账号，你们的基础很好，就缺…" |
| **G2 诊断** | 用专业术语和框架 | 用 Clover/SPIN 的框架，而不是漫无目的地提问 |
| **G5 价值展示** | 展示专业性和专注 | "我们只服务餐饮行业，做精不做广。所以我们的方法对你们很匹配" |
| **异议处理** | 用数据和权威来回应 | "这个问题我们已经见过 200+ 次，通常是因为…" |

**为什么有效**：

Milgram 的著名实验：
一个穿着白大褂（权威象征）的人说"继续"，普通人会持续按高达致命电压的按钮（实际没有真实电击，但参与者不知道）。换掉白大褂，服从率大幅下降。

销售中的应用：
```
销售说："我觉得你们这样不太对"
→ 客户想："你凭什么说我不对"（防御）

销售说："在我们服务的 500+ 企业中，有 80% 一开始都这样做，
后来改成了…方法，数据提升了…"
→ 客户想："哦，原来这么多人都这样，那我应该也这样"（信任）
```

**建立权威性的三个方式**：
1. **头衔 + 经验**："我在这个行业做了 10 年，"
2. **数据 + 证明**："服务了 500+ 企业，92% 续费率"
3. **专注 + 深度**："我们只做内容运营，不做其他，所以特别专业"

### 原理 5：喜爱度（Liking）

**心理学解释**：
人更容易从喜欢的人那里买东西。喜爱度取决于：外表、相似性、奉承、合作。

**在销售中的应用**

| 环节 | 应用方式 | 话术例子 |
|------|---------|---------|
| **G1 破冰** | 建立相似性 | "我也是从电商出身的，所以很理解你们的…" |
| **G1 破冰** | 镜像和匹配（Sandler） | 客户说快，你就快回应；客户慢，你就慢 |
| **全程** | 真诚的奉承 | "你对这块业务的理解真深" （必须真诚，假的会被看出） |
| **G7 跟单** | 建立个人关系 | "我最近看了一篇你们行业的文章，很有启发…" |

**为什么有效**：

Cialdini 的"两个陌生人实验"：
- A 组：两个陌生人在一个房间里讨论一个问题（他们被告知对方是来自相同背景的）
- B 组：两个陌生人被告知对方来自不同背景

结果：A 组的人相互喜欢程度高 2 倍。仅仅是"相似性认知"就足以提升喜爱度。

销售中的应用：
```
销售说："我们的服务怎么怎么好"
→ 客户评估的是：销售有没有真诚理解我

销售说："我看过你们的账号，内容质量真的不错。
我最早也是这样做的，后来遇到了…问题"
→ 客户想："这个人理解我，我喜欢他"
```

**提升喜爱度的最强技巧**（来自销售心理学研究）：
不是"讲你有多厉害"，而是"让客户感觉你理解他"

### 原理 6：稀缺性（Scarcity）

**心理学解释**：
人的大脑对"失去"的反应，强于对"获得"的反应。稀缺性激发了"害怕失去"的情绪。

**在销售中的应用**

| 环节 | 应用方式 | 话术例子 |
|------|---------|---------|
| **G1 破冰** | 建立独特性 | "我们只和 20 家企业合作，做精不做广" |
| **G5 价值展示** | 创造稀缺感 | "我们这个月只有两个名额，下月会涨价" |
| **G6 收口** | 推进成交 | "我这边有个名额能给你留着，但需要今天确认" |
| **G7 跟单** | 时间限制 | "这个优惠仅限本周，下周恢复原价" |

**为什么有效**：

Cialdini 的"芝加哥煎饼房实验"：
- A 组：被告知"这个菜很受欢迎" → 购买意愿 60%
- B 组：被告知"这个菜很受欢迎，而且最近原料难获得，可能很快就下架" → 购买意愿 90%

销售中的应用：
```
销售说："我们的方案提供了…"
→ 客户想："不急，我先看看有没有更好的"

销售说："我们这个月只有三个名额。
你要是感兴趣，我建议今天就锁定，
否则下月要等，而且下月价格会调整"
→ 客户想："哦，有人在排队，我得快点"
```

**稀缺性的三个形式**：
1. **数量稀缺**："我们只能服务 20 家"
2. **时间稀缺**："这个优惠截止到本周五"
3. **信息稀缺**："只有前 10 个客户能看到这个方案"（内测、提前价等）

## 6 大原理在 Clover 各环节的完整应用

### G1 破冰

```
目标：建立喜爱度和权威性

使用的心理学原理：
1. 喜爱度 - 建立相似性
   "我也是做…的，所以很理解你们"
   
2. 权威性 - 展示有备而来
   "我看了你们的账号，[具体数据]，基础很好"
   
3. 互惠原则 - 先给价值
   "我整理了一份竞品分析，先给你参考"

话术框架：
"XX 总，感谢今天的时间。我提前看了你们的[具体成就]，
[展示权威] 你们在这块其实是有基础的。
今天我想诊断一下具体情况，看我们能不能帮上忙——
帮不上的话我也会直说。[建立信任]"

心理学效果：
✓ 喜爱度高 → 客户放松
✓ 权威性强 → 客户认可你的专业
✓ 互惠感→ 客户愿意敞开心扉
```

### G2 诊断

```
目标：让客户自己说出痛点（承诺原则）

使用的心理学原理：
1. 承诺与一致性 - 让客户说出痛点
   Q: "你们试过…吗？为什么停下来了？"
   A: 客户说出了问题
   现在客户已经在"承诺"问题的存在
   
2. 权威性 - 用框架和专业术语提问
   不是漫无目的，而是用 Clover/SPIN 的框架
   
3. 社会证明 - 引入"我们见过"
   "这个问题我见过 100+ 次，通常原因是…"

话术框架：
"你们现在[情景题]？[停顿，让客户说]
这样会不会遇到[困难题]？
[关键]："如果问题不解决，会对你们怎样？"
← 这一步让客户自己量化问题的影响

心理学效果：
✓ 承诺原则 → 客户说了，就会相信这是个问题
✓ 权威性 → 框架性的提问看起来很专业
✓ 社会证明 → "我们见过…" 增加信任度
```

### G3 极限拉扯

```
目标：放大问题和欲望（稀缺性 + 对比效应）

使用的心理学原理：
1. 对比效应 - "现在 vs 理想"的巨大差距
   "现在是[负面]…理想是[正面]……差距多大？"
   
2. 稀缺性 - 时间稀缺和机会稀缺
   "如果现在不改，明年和竞品的差距会怎样？"
   "你现在知道有这样的可能性，但不做的话…"
   
3. System 1 激发 - 情绪而非逻辑
   放大的是"害怕失去"和"渴望得到"
   不是理性论证

话术框架：
向下拉扯（激发害怕失去）：
"你现在每月花…成本。如果继续…，明年累计浪费多少？
竞品在[加速]，你们在[停滞]，这个差距怎么办？"

向上拉扯（激发渴望得到）：
"想象解决这个问题后，你能…？
那对整个公司的影响有多大？"

心理学效果：
✓ 稀缺性 → 客户感受到"不做就会失去"
✓ 对比效应 → 现在和理想的 Gap 越来越明显
✓ System 1 激活 → 情绪承诺已经发生（在逻辑之前）
```

### G4 承诺

```
目标：让客户说出"现在就要改变"（承诺原则的巅峰）

使用的心理学原理：
1. 承诺与一致性 - 小步承诺法则
   每一个小"是"都强化了客户的身份认同
   到最后，客户已经"承诺"了要改变
   
2. 损失厌恶 - 强化"不改变的代价"
   "如果继续用现在的方式，这个 Gap 能自己解决吗？"
   ← 客户的答案会是"不能"，强化了改变的必要性

话术框架：
Step 1：总结归因（让客户确认）
"所以你的情况是…这个问题会对你怎样？"
← 客户说"是的"→ 承诺原则激活

Step 2：扩大差距（强化对比）
"如果继续用现在的方式，半年后会怎样？"
← 损失厌恶激活

Step 3：获取承诺（最关键）
"那你是想找到方法现在解决，还是继续这样？"
← 客户要么说"现在解决"（成交概率 80%+）
   要么说"还要再想想"（成交概率 30%以下）

心理学效果：
✓ 承诺原则 → 客户已经"说出了"改变的意愿
✓ 一致性 → 客户会为了与自己的话保持一致而行动
✓ 损失厌恶 → 强化了不改变的代价
```

### G5 价值展示

```
目标：展示"解决方案"而不是"产品功能"（权威性 + 社会证明）

使用的心理学原理：
1. 社会证明 - 用"和你们一样的企业"的案例
   "跟你们情况类似的电商企业…"
   ← 比"我们的客户…"有 5 倍的说服力
   
2. 权威性 - 展示专业和专注
   "我们只做内容运营，不做其他，所以…"
   
3. 故事优于数据 - 用故事激活 System 1
   不是"我们提升了 20% 转化率"
   而是"有一家企业，他们…后来…结果…"

话术框架：
① 定位专家身份（权威性）
"我们只服务[垂直行业]，做精不做广。
所以我们的方法对你们是匹配的。"

② 用故事展示价值（社会证明 + System 1）
"跟你们类似的企业，以前也是[你现在的状态]。
后来他们改变了…[故事]
结果…[具体数据]"

③ 强调交付结果（不是产品）
"你不需要操心实现，你只需要…[你的工作]。
我们负责…[我们的承诺]"

心理学效果：
✓ 社会证明 → "他们和我一样，他们行，我也行"
✓ 权威性 → 专业的展现增加信任
✓ System 1 激活 → 故事让人记得更清楚，说服力更强
✓ 降低交付风险 → 明确的承诺减少了客户的"害怕失去"
```

### G6 成交收口

```
目标：从"理性同意"变成"行动承诺"（稀缺性 + 承诺原则）

使用的心理学原理：
1. 稀缺性 - 创造时间紧迫感
   "我这边还有一个名额，但需要…"
   
2. 承诺原则 - Sandler 的小步成交
   一步步让客户说"是"
   
3. 现状偏差的反制 - 强调改变的收益 > 风险
   "现在不改的风险 vs 改的成本"

话术框架：
① 报价
"我们的方案是…每月 X 元，包含…"
[停顿 → 什么都不说，等客户开口]

② 处理异议（用 Sandler 反向）
客户："价格太贵"
销售："价格贵相对什么说——相对一个全职员工，还是…？"
← 把决策权球踢回给客户

③ 小步成交（Sandler）
"第一步，你同意我们先做…诊断吗？"
客户："可以"
"很好。诊断结果出来，我们一起看，对吗？"
…[逐步推进]

④ 最终承诺
"那我们现在签这份协议，付这个定金，可以吗？"

心理学效果：
✓ 承诺原则 → 每个"是"都推动下一个"是"
✓ 稀缺性 → 时间压力 + 名额限制激发购买欲
✓ 现状偏差被克服 → 强化了改变的必要性 > 改变的风险
```

### G7 跟单系统

```
目标：维持承诺一致性，从"签单"到"用起来"（互惠原则 + 一致性）

使用的心理学原理：
1. 互惠原则 - 不催单，先送价值
   D1 送诊断报告
   D3 送行业触点
   D5 送案例
   ← 每一次都强化互惠感
   
2. 承诺一致性 - 强化客户已经的"是"
   "你上次说想…，现在有进展吗？"
   ← 提醒客户他已经承诺了什么
   
3. 社会证明 - 定期送"别人在用"的证据
   "我们又一个客户…"
   ← 强化"别人都在做，你不做会落后"

话术框架：
D1：感谢 + 诊断价值
"今天聊了很多，我整理了竞品分析…"

D3：行业触点
"你们行业昨天有…我觉得和你们很匹配…"

D5：案例证明（社会证明）
"我们客户的内容本周审批了，效果…"

D7：轻触收口
"大概一周了，你那边有什么新想法吗？我这边的名额还留着"

心理学效果：
✓ 互惠感 → 客户欠债感越来越强，越来越想"还债"（成交）
✓ 承诺一致性 → 不断提醒，强化已有承诺
✓ 社会证明 → 定期刷存在感，强化"别人都在做"
```

## 综合案例：从 System 1 到 System 2

### 一通完整通话的心理学分解

```
【通话开始 - System 1 激活】

销售 G1："我看了你们的账号，基础很好"
← 权威性：展示有备而来
← 喜爱度：建立相似性（做内容的人）

客户反应：放松，开始讲他的故事
心理学：System 1 被激活，情感通道打开

【G2 诊断 - 承诺原则开始】

销售："这样做多久了？遇到什么困难？"
← 让客户说出问题

客户："我们员工说太累了，所以停了"
心理学：客户已经在"承诺"问题的存在

【G3 拉扯 - System 1 强化】

销售："竞品这个月又新增了 1 万粉，你们现在停更了，
        这个差距明年会怎样？"
← 稀缺性：时间窗口在关闭
← 对比效应：现在 vs 理想的巨大差距

客户："天哪，我们会被甩开……"
心理学：情绪波动强烈，System 1 的信号非常强

【G4 承诺 - 一致性原则激活】

销售："那你是现在想解决，还是再看看？"

客户："现在肯定要解决……"
心理学：客户已经说出了改变的承诺
← 承诺原则激活：一旦说出来，就会努力保持一致

【G5 价值 - 社会证明强化】

销售讲一个故事："有个餐饮企业也停更过，后来…"
← 社会证明：和你一样的人在做，而且效果好

客户："是啊，我们行业确实有这样的趋势…"
心理学：System 1 已经完全同意，System 2 开始找支持的理由

【G6 收口 - Sandler 小步承诺】

销售："第一步，你同意我们先诊断吗？"
客户："可以啊"

销售："很好。诊断后我们一起看，对吗？"
客户："对"

销售："如果诊断显示有 3 个以上的优化空间，你同意试用 30 天吗？"
客户："可以"

销售："那现在我们签这份协议，付定金，对吗？"
客户："…好吧"
心理学：多个小"是"累加，最后的大承诺变成顺势而为

【成交后 - 一致性维护】

现在客户已经说出了这些"是"：
✓ "这确实是个问题"（G2 承诺）
✓ "现在很紧急"（G3 承诺）
✓ "我想改变"（G4 承诺）
✓ "我同意试用"（G6 承诺）

客户的整个自我认知都在说："我是一个正在解决问题的人"
客户的 System 2 现在只会找支持这个"是"的理由

即使中间有疑虑，一致性原则也会推动他继续推进（而不是反悔）
```

## 心理学陷阱（要避免的）

### ❌ 错误 1：太早启动 System 2

```
错误做法：
"让我给你展示一下数据和 ROI 计算…"
→ 启动 System 2（逻辑分析）
→ 客户开始反驳："但是…我们的情况不同"

正确做法：
先用故事、案例、对比激发 System 1 的情绪
让情绪承诺先发生
System 2 只会找支持而不是反驳的理由
```

### ❌ 错误 2：跳过互惠原则

```
错误做法：
一上来就问"你们有没有预算"
→ 客户警觉：这是销售在套我

正确做法：
先给价值（D1 诊断报告、D3 行业触点）
建立互惠感
客户会自动愿意"还债"（成交）
```

### ❌ 错误 3：社会证明用错了

```
错误做法：
"我们有 500+ 客户，客户满意度 98%…"
→ 客户想："这和我有什么关系"

正确做法：
"跟你们一样是 ¥2000 万规模的电商企业，他们用了我们后…"
→ 客户想："他们和我一样，他们行，我也行"
```

### ❌ 错误 4：承诺原则用错了方向

```
错误做法：
销售："你同意我们是解决方案吗？"
→ 客户可能说"不同意"

正确做法：
销售："你同意这个问题存在吗？"（G2）
销售："你同意这很紧急吗？"（G3）
销售："你同意现在想改变吗？"（G4）
← 每一步都让客户说"是"到"改变"，而不是"是"到"我们的产品"
```

## 心理学检查清单

每通通话后，问自己：

- [ ] **G1 - 我建立了喜爱度（相似性）吗？**
- [ ] **G1 - 我展示了权威性吗？**
- [ ] **G2 - 我让客户"承诺"了问题的存在吗？**
- [ ] **G3 - 我激发了 System 1 的情绪反应吗？**
- [ ] **G3 - 我用对比效应放大了 Gap 吗？**
- [ ] **G4 - 我让客户说出了"改变意愿"吗？**
- [ ] **G5 - 我用了"和你一样的企业"的社会证明吗？**
- [ ] **G5 - 我用故事而不是数据展示价值吗？**
- [ ] **G6 - 我用小步承诺推进成交吗？**
- [ ] **G6 - 我创造了稀缺性（时间紧迫）吗？**
- [ ] **G7 - 我用互惠原则维持客户承诺吗？**

如果有 3 个以上的"否"，那这通电话就有明显的改进空间。
