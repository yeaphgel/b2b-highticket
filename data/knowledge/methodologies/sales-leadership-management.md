# 销售领导力与团队管理

**核心理论**：Mark Roberge《The Sales Acceleration Formula》+ Jason Jordan《Cracking the Sales Management Code》 | **适用**：销售团队管理、复制成功个体 | **核心问题**：如何把"超级销售"的能力复制给整个团队？

---

## 核心洞察：好的销售团队不是"招好销售"，而是"建好系统"

### 为什么大多数销售团队做不大？

**错误做法**：
```
"我需要找更多好销售"
→ 好销售很难招
→ 好销售不稳定（会走）
→ 好销售的经验在个人脑子里，不在团队系统里
```

**正确做法**：
```
"我需要建一套系统，让普通销售也能产生好结果"
→ 系统化的流程、话术、工具
→ 任何人都可以培训
→ 经验在系统里，不在个人脑子里
```

**Mark Roberge 的研究**（HubSpot 从 0 到 $100M 的销售副总裁）：
```
成功的销售团队有 5 个要素：
1. 招对人（Hire the Same Person）
2. 培训好（Train the Same Way）
3. 流程清（Manage the Same Process）
4. 目标明（Set the Same Goals）
5. 数据准（Measure the Same Metrics）
```

## 销售领导力的 5 要素

### 要素 1：招对人（Hire the Right Profile）

**传统招聘的陷阱**：
```
"我要找一个有 10 年经验的销售"
→ 经验不代表能力
→ 10 年坏习惯可能比 0 经验更糟
```

**销售成功画像（Sales Success Profile）**：
```
不要看：工作经验年限、毕业学校
要看：
1. 可辅导性（Coachability）—— 能否接受反馈并改变？
2. 好奇心（Curiosity）—— 是否主动了解客户？
3. 内驱力（Intrinsic Motivation）—— 为什么想做销售？
4. 智能（Intelligence）—— 能否快速学习？
5. 毅力（Work Ethic）—— 能否坚持？
```

**面试中的 Coachability 测试**：
```
1. 给候选人一个销售场景，让他演示话术
2. 给出具体的改进反馈："试试换个方式说 XX"
3. 让他再演示一次
4. 观察：他是否吸收了反馈？改变了多少？

A+ 候选人：立刻调整，改进明显
B 候选人：尝试改变但不够到位
C 候选人：不接受反馈或改不了
→ 只招 A+ 和 B
```

**Clover 中的应用**：
```
招新销售时，让他模拟 Clover G1-G3 的对话：
1. 观察他是否先建立连接再诊断
2. 给他反馈："试试用 SPIN 方式提问"
3. 看他能否快速调整
```

---

### 要素 2：系统化培训（Train the Same Way）

**为什么培训要系统化？**
```
个人经验式培训：
"我教你我怎么做" → 每个销售学到的是不同的东西

系统化培训：
"这是标准流程，每个人按这个来" → 所有销售有一致的能力
```

**Clover 销售团队培训体系**：

```
Week 1：Clover 基础
  - 1-7-3 节奏理解
  - G1-G7 每个环节的目标和话术
  - 角色扮演：模拟 G1 破冰

Week 2：销售方法论
  - SPIN 问诊法训练
  - Sandler 资格审查
  - 角色扮演：模拟 G2-G3 诊断

Week 3：实战演练
  - 跟随资深销售旁听真实通话
  - 记录每个环节的表现
  - AI 复盘：录音转文字，用 Clover 框架打分

Week 4：独立上岗
  - 开始独立接客户
  - 每周 1 对 1 复盘
  - 持续改进
```

**培训成功的指标**：
```
- 新销售第 1 个月能完成 G1-G3 的标准化对话
- 第 2 个月能独立完成 G4-G6 的成交流程
- 第 3 个月能达到团队平均成交率的 70%
```

### 要素 3：管理销售流程（Manage the Process）

**为什么不能"让销售自由发挥"？**
```
自由发挥的结果：
- 每个销售用不同的流程
- 有的销售跳过诊断直接报价
- 有的销售忘了跟单
- 管理者无法标准化改进
```

**标准化的 Clover 销售流程**：

```
Pipeline 阶段：
1. 线索 → 确认 ICP 符合 → PRE 筛选通过
2. 初次接触 → 完成 G1 破冰 + G2 诊断
3. 深度诊断 → 完成 G3 拉扯 + G4 承诺
4. 方案演示 → 完成 G5 价值展示
5. 谈判 → 完成 G6 收口（收到意向金/签约）
6. 成交 → 启动 G7 跟单
7. 培育 → 30 天后未成交，进入长期培育

每个阶段的标准：
- 必须有明确的进入和退出条件
- 每个阶段有预计时长
- 每个阶段有标准产出（诊断报告、方案书等）
```

**Pipeline 管理会议（每周）**：
```
1. 每个销售的 Pipeline 数据：
   - 本周新增多少线索？
   - 多少进入诊断阶段？
   - 多少进入方案阶段？
   - 多少成交？
   - 多少流失？

2. 重点客户回顾：
   - "张总这个客户现在在哪个阶段？"
   - "下一步是什么？"
   - "有什么障碍？"

3. 改进项：
   - 哪个阶段转化率最低？
   - 需要什么培训或工具来改善？
```

### 要素 4：设定清晰目标（Set Clear Goals）

**目标设定的原则**：
```
不是"这个月多签几单"
而是"每个阶段的转化率提升 X%"
```

**销售目标的 3 层结构**：

| 层级 | 指标 | 例子 |
|------|------|------|
| **结果指标** | 签单金额/数量 | 本月签 5 单，收入 ¥15 万 |
| **过程指标** | 各环节转化率 | G1→G2 转化率 60% |
| **活动指标** | 每日行动量 | 每天 10 通电话，3 个诊断 |

**为什么过程指标最重要**：
```
结果指标（签单数）是"滞后指标"——月底才知道
过程指标（转化率）是"领先指标"——每周可以调整

如果 G1→G2 转化率只有 30%：
→ 问题在破冰环节
→ 需要加强 G1 培训
→ 月底才不会"突然发现目标没完成"
```

---

### 要素 5：数据驱动决策（Measure with Data）

**销售管理需要关注的 5 个核心数据**：

| 数据 | 含义 | 健康标准 |
|------|------|---------|
| **线索→诊断率** | 破冰能力 | ≥ 40% |
| **诊断→方案率** | 需求挖掘能力 | ≥ 50% |
| **方案→成交率** | 成交能力 | ≥ 30% |
| **平均成交周期** | 销售效率 | ≤ 21 天 |
| **客户留存率** | 客户成功 | ≥ 85%/年 |

**数据驱动的销售改进**：
```
问题：本月签单量下降了 30%

错误做法：
"大家再努力一点，多打电话"

正确做法：
"看数据——线索量没变，但线索→诊断率从 50% 降到 35%。
  问题在破冰环节。
  我抽查了 5 通 G1 录音，发现 3 个问题：
  1. 开场没有个性化（直接介绍公司）
  2. 没有展示有备而来
  3. 没有降低客户防御心
  
  下周的重点：重新培训 G1 破冰，每通 G1 录音都要复盘。"
```

## 销售团队管理的常见错误

### ❌ 错误 1：用"激励"代替"系统"

```
"这个月签单最多的奖 ¥5000！"
→ 短期有效，但问题没解决

正确做法：
→ 找到为什么签单少（看数据、查流程）
→ 提供培训和工具
→ 建立可复制的系统
```

### ❌ 错误 2：用"平均"评估个人

```
"你的转化率低于团队平均，要多努力"
→ 不是"努力"的问题，是"方法"的问题

正确做法：
→ 分析哪个环节低于平均
→ 针对性培训那个环节
→ 让高转化率的同事分享话术
```

### ❌ 错误 3：只关注"签单"不关注"留存"

```
销售签了单就不管了
→ 客户 3 个月后流失
→ 下个月又要花 CAC 找新客户

正确做法：
→ 销售的 KPI 中包含留存率
→ 成交后 G7 跟单要标准化
→ 客户成功团队要及时跟进
```

## 销售领导力与 Clover 的融合

### Clover 团队运营框架

```
标准化：
- 所有销售用同一个 Clover 1-7-3 流程
- G1-G7 每个环节有标准话术和通关标准
- 每个客户的跟进记录存在系统中（data/clients/）

可复制：
- 新销售培训用 Clover 方法论体系
- 每周 AI 复盘：录音转文字，Clover 框架打分
- 每月方法论更新：根据实战数据改进

可管理：
- 每周 Pipeline 会议：按 Clover 阶段跟踪
- 每个销售的各环节转化率看板
- 问题环节 → 针对性培训
```

## 销售领导力快速检查清单

作为销售团队领导者，每周问自己：

- [ ] **招聘**：我们的成功画像更新了吗？最近面试的人 Coachability 如何？
- [ ] **培训**：新销售能否独立完成 G1-G6 全流程？
- [ ] **流程**：每个销售都在按标准流程走吗？有没有跳过环节的？
- [ ] **目标**：团队的目标清晰吗？每个人知道自己的 KPI 吗？
- [ ] **数据**：哪个阶段的转化率最低？下周要重点改进什么？
- [ ] **留存**：成交客户的留存率健康吗？G7 跟单在标准化执行吗？

如果以上有 3 个以上的"否"，你的团队存在系统性问题。
